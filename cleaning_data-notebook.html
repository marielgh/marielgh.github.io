Combining Datasets¶

Globbing & concatenate

import glob

csv_files = glob.glob('*.csv')

frames = []
for csv in csv_files:
    df = pd.read_csv(csv)
    frames.append(df)

uber = pd.concat(frames)

pd.concat concatenates per default vertically (adding rows) (axis=0)

If we want to concatenate horizontally, add axis=1

Merging

pd.merge function

pd.merge(left, right, how='inner', on=None, left_on=None, right_on=None,
         left_index=False, right_index=False, sort=True)

Dataframe.merge function

df.merge(right, how='inner', on=None, left_on=None, right_on=None, 
         left_index=False, right_index=False, sort=False)

how: default ‘inner’

  • left: use only keys from left frame, similar to a SQL left outer join; preserve key order
  • right: use only keys from right frame, similar to a SQL right outer join; preserve key order
  • outer: use union of keys from both frames, similar to a SQL full outer join; sort keys lexicographically
  • inner: use intersection of keys from both frames, similar to a SQL inner join; preserve the order of the left keys

def check_null_or_valid(row_data):
    """Function that takes a row of data, drops all missing values,
    and checks if all remaining values are greater than or equal to 0
    """
    no_na = row_data.dropna()
    numeric = pd.to_numeric(no_na)
    ge0 = numeric >= 0
    return ge0
